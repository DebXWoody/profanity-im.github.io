Joining a chat room

Use the /join command to join a chat room supplying the name of the room and the nickname you wish to be known as in the room:

/join room@conference.server.com mynick

The nickname parameter is optional, if not supplied the 'localpart' of your Jabber ID will be used. For example if you are logged in as itsme@server.com the following command will enter a room with the nickname itsme:

/join room@conference.server.com

You may omit the domain part of the chat room address if it is the default conference.domain, for example, if you are logged in as kai@mycompany.org, using the following command:

/join development

is the same as:

/join development@conference.mycompany.org

The default nickname and conference service may be changed for any account using the /account command.

Messages entered in the input bar will be sent to the room.

Changing your nickname

Use the /nick command whilst in a chat room to change your nickname, for example to change your nickname to bob:

/nick bob

Listing room participants

Room participants can be listed at any time whilst in a chat room by using the /who command.

The /who command takes an optional parameter specifying the presence you are interested in:

chat
online
away
xa
dnd
available
unavailavle

Inviting contacts to a room

To invite one of your contacts to the current chat room, use the /invite command with an optional reason for the invite:

/invite Bruce

or with a reason:

/invite Bruce Come and join the chat

As with other commands, the name must be double quoted if it contains spaces:

/invite "Bruce Dickenson" Join us!

Sending a private message

To start a private chat session with a room participant use the /msg command supplying the participants nickname:

/msg "Some User" Hello there!
/msg mike hello there!

Leaving a room

To leave a chat room, use the /leave command, the /close command has the same effect.
