Joining a chat room

Use the /join command to join a chat room supplying the full name of the room and the nickname you wish to be seen as in that room:

/join room@conference.server.com mynick

The nickname parameter is optional, if not supplied the starting part of your Jabber ID will be used. For example if you are logged in as itsme@server.com the following command will enter a room with the nickname 'itsme':

/join room@conference.server.com

Changing your nickname

Use the /nick command whilst in a chat room to change your nickname, for example to change your nickname to 'bob':

/nick bob

Listing room participants

Room participants can be listed at any time whilst in a chat room by using the /who command.

Sending a private message

To start a private chat session with a member of the chat room, use the /msg command with the full Jabber ID of the room member (including the room name):

/msg room@conference.server.com/someuser

Leaving a room

To leave a chat room, use the /close command whilst the room is visible in the main window area.
