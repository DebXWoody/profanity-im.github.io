Overview

In theory Profanity should run on any POSIX complaint operating system, meaning GNU/Linux and any other *nix variants such as OS X.

To run on Windows, a POSIX environment is required, such as Cygwin.

Arch and Gentoo currently have community supported packages.

Until distribution packages are created, a simple install script has been written which will work on Ubuntu and Fedora, the script simply installs dependencies and runs the usual ./configure && make && make install

For all other distributions/operating systems the install script can be used as a guide to installing dependencies.

Installing on Linux

Arch

Profanity is available in the AUR:

http://aur.archlinux.org/packages.php?ID=63780

Instructions for installing from the AUR are at:

https://wiki.archlinux.org/index.php/AUR_User_Guidelines#Installing_packages

Gentoo

Profanity is available from the 'stuff' overlay. Too install:

Add the "stuff" overlay:

layman -a stuff

or add to existing:

layman -s stuff

Install the package:

emerge profanity

Profanity can be linked with either libxml2 or expat libraries. By default expat is used, set the 'xml' flag to link with libxml2.

To install latest develepment version, add net-im/profanity to package.keywords file:

echo "=net-im/profanity-9999 **" >> /etc/portage/package.keywords
emerge =profanity-9999

Ubuntu

Download the latest source tarball from the homepage.

Extract the file and cd into the directory e.g:

tar -xvf profanity-0.1.10.tar.gz
cd profanity-0.1.10

Run the installer:

./install-all.sh

Fedora

If your user doesn't have Sudoer rights, add them to the list:

su
<enter password>
visudo

Find the line:

root ALL=(ALL) ALL

And add your user under it:

<username> ALL=(ALL) ALL

Download the latest source tarball from the homepage.

Extract the file and cd into the directory e.g:

tar -xvf profanity-0.1.10.tar.gz
cd profanity-0.1.10

Run the installer:

./install-all.sh

Other distros

For other linux distributions, the install script is a good guide. You will need to install the development packages for the dependencies, which are:

libssl, expat, ncurses, glib2.0, libnotify, libcurl and libXScrnSaver.

Profanity uses the libstrophe XMPP library which must be downloaded and built from the source which can be found at Github.

Installing on Mac OS X

Installing on Windows

You will need Cygwin to run profanity on Windows:

http://www.cygwin.com

If you don't already have it, install Cygwin by downloading and running:

http://cygwin.com/setup.exe

You will need git, wget and unzip to install Profanity which are not by default installed with Cygwin. When running setup.exe (or running it again to install these packages) you can search for them and add them to the install when you are asked to select packages.

Download the latest source zip from the homepage

Put the zip somewhere on your Cygwin drive for example:

C:\cygwin\home\james

Unzip the file and cd into the directory e.g:

unzip profanity-0.1.10.zip
cd profanity-0.1.10

Run the installer:

./install-all.sh
