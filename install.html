Switching between virtual windows

To make a virtual window visible in the main window area, use Alt-1 to Alt-0 or the F1 to F10 keys. Certain commands automatically make a virtual window visible such as joining a chat room, or sending a message to a contact.

When a window is visible new messages within it do not highlight the windows number in the status bar.
