Creating a Theme

To create a theme, create a file with the theme settings and put it in the themes directory.

To list available themes:

/theme list

To enable the theme:

/theme <theme-file>

See the /theme command.

Example Themes

Example themes can be found in the themes folder of the source code.

Available colours are black, white, red, green, blue, yellow, cyan, magenta and default, where default is the default foreground or background colour on your terminal.
