prof module¶

Profanity plugin API

prof.cons_alert()¶

Highlights the console window in the status bar.

prof.cons_bad_cmd_usage(command)¶

Show a message indicating the command has been called incorrectly.

Parameters:command – the command name with leading slash, e.g. "/say"
prof.cons_show(message)¶

Show a message in the console window.

Parameters:message – the message to print
Returns:True on success, False on failure
prof.cons_show_themed(group, item, default, message)¶

Show a message in the console, using the specified theme.

Themes can be must be specified in ~/.local/share/profanity/plugin_themes

Parameters:
  • group – the group name in the themes file, or None
  • item – the item name within the group, or None
  • default – default colour if the theme cannot be found, or None
  • message – the message to print
Returns:

True on success, False on failure
