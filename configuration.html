Configuring Profanity

It is recommended to use the commands provided within Profanity to make configuration changes rather than directly editing the configuration file.

The following shows an example configuration file, and lists below the commands related to each property.

Example Configuration File

[ui]
theme=boothj5
beep=false
flash=false
intype=true
splash=true
history=true
vercheck=true
titlebar.version=false

[notifications]
message=true
typing=true
remind=20

[chatstates]
enabled=true
outtype=true

[logging]
maxsize=1048580
chlog=true

[connection]
reconnect=30
autoping=120

[presence]
priority=0
autoaway.mode=away
autoaway.time=20
autoaway.message=I'm away from my computer
autoaway.check=true
