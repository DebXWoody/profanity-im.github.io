Configuring Profanity

It is recommended to use the commands provided within Profanity to make configuration changes rather than directly editing the configuration file.

The following shows an example configuration file, and lists below the commands related to each property.

Example Configuration File

[ui]
flash=false
splash=true
beep=false
theme=boothj5
history=true
intype=true
mouse=false
statuses=true
titlebar.version=true
vercheck=true

[notifications]
message=true
typing=true
invite=true
sub=true
remind=30

[chatstates]
gone=5
enabled=true
outtype=true

[logging]
chlog=true
grlog=true
maxsize=2048580

[connection]
autoping=120
reconnect=60
account=jabber.org

[presence]
autoaway.mode=away
autoaway.time=10
autoaway.message=I'm not here
autoaway.check=true
