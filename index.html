Profanity

A console based XMPP client

Profanity is a console based XMPP client written in C using ncurses and libstrophe, inspired by Irssi

Latest release: 0.1.10
Downloads: profanity-0.1.10.tar.gz
profanity-0.1.10.zip

Profanity will run on Linux, Mac OS X and Windows (using Cygwin).

See the User Guide for information on installing and using Profanity.

Mailing list: Google Groups Profanity

The source can be found on Github

Unicode support is currently available using latest development at Github and will be in the next release.

Follow @ProfanityIM Tweet

Features

  • Supports most XMPP chat services, including GTalk, Facebook
  • Command driven user interface
  • Options to customise functionality and user interface
  • Desktop notifications
  • Chat logging
  • Chat rooms and private messaging
  • Ability to send tiny urls
  • Support for simple colour themes
