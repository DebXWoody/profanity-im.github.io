Profanity

A console based XMPP client

Profanity is a console based XMPP client written in C using ncurses and libstrophe, inspired by Irssi

Latest release: 0.3.1
Downloads: profanity-0.3.1.tar.gz
profanity-0.3.1.zip

Profanity will run on Linux, Mac OS X and Windows (using Cygwin).

See the User Guide for information on installing and using Profanity.

Read the mailing list at Google Groups Profanity, follow at Google+ and Twitter, or view the source at Github

Join the chatroom at profanitydev@conference.jabber.org.

See How to help out if you would like to get involved.

Currently in development Off The Record (OTR) messaging, and plugins with support for C, Python, Ruby and Lua.

Features

  • Supports XMPP chat services, including GoogleTalk and Facebook
  • Command driven user interface
  • Options to customise functionality and user interface
  • Desktop notifications
  • Chat logging
  • Chat rooms and private messaging
  • Roster management
  • Flexible resource and priority settings
  • Ability to send tiny urls
  • Full unicode support
