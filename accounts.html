Configuring Accounts

The /account command should be used to modify the settings below.

The following is an example account configuration element

[work]
enabled=true
jid=some.user@chatty.com
server=main.chatserv.org
resource=profanity
presence.last=away
presence.login=dnd
priority.online=20
priority.chat=30
priority.away=10
priority.xa=0
priority.dnd=-1
