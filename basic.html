Starting Profanity

To start Profanity, run:

$ profanity

From your command line.

User Interface

The user interface is split into four main areas, from top to bottom:

  • Title bar
  • Main window area
  • Status bar
  • Input area

The title bar shows the title for the current visible 'virtual window' (usually the contact you are chatting with, or the chat room you have joined), and your current presence in the far right corner.

The main window area is used to show one of up to 10 virtual windows, numbered 1,2,3,4,5,6,7,9,0. The first window (1) is always used as the 'console' window where system messages and command responses are shown. The remaining windows (2 to 0) are available for chatting with contacts, and joining chat rooms.

The status bar shows the time, the user you are logged in as, and on the far right the status of each virtual window. If the virtual window is used, its number will appear, if it has new messages of any kind, the number will be highlighted.

The input area is where you enter all of your commands and messages. When in the console, commands may be entered here, when in chat with a user or a chat room, both commands and messages may be entered.

Switching between virtual windows

To make a virtual window visible in the main window area, use Alt-1 to Alt-0 or the F1 to F10 keys. Certain commands automatically make a virtual window visible such as joining a chat room, or sending a message to a contact.

When a window is visible new messages within it do not highlight the windows number in the status bar.

Connecting

To connect to an XMPP server, use the /connect command:

/connect user@server.com

You will be prompted by the status bar to enter your password. Once entered you will be connected, and you should start receiving presence notifications from your contacts in the console window.

Sending a message

To send a message use the /msg command:

/msg mycontact@server.com Hello there!

The above will send "Hello there!" to mycontact@server.com. A new virtual window will be openend and become visible in the main window area, the message will be sent and displayed.

The message is optional:

/msg mycontact@server.com

The above will open the new window, make it visible, but no message is sent. This can be useful if you don't currently have a virtual window open with the contact and want to see some history of your conversation with that contact (see the /chlog and /history settings).

Closing a virtual window

You can close a virtual window whilst viewing it by using the /close command.

Listing virtual windows

To see a list in the console of all your currently used virtual windows, what they are being used for and whether there are any unread messages in them, enter the following command:

/wins

Changing your presence

To change your presence, use the following commands:

/away
/xa
/chat
/online
/dnd

Each of these commands accepts a message as a parameter, so to show the message "I'm very busy" to your contacts when setting status to Do Not Disturb:

/dnd I'm very busy

Logging out

To log out from the current server, use the /disconnect command.
